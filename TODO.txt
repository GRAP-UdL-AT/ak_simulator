Environment creation
-----------------------
python -m venv testing_package_venv
pip install --upgrade pip

Testing package
---------------------
twine upload --repository testpypi ./dist/*
pip install -i https://test.pypi.org/simple/ ak-simulator


Final repository
---------------------
twine upload ./dist/* --verbose
pip install ak-simulator


Local installation
------------------
pip install ak_simulator-0.0.7-py3-none-any.whl
pip install ak_simulator-0.0.7-py3-none-any.whl --force-reinstall

Execution
------------------
python -m ak_simulator


Installation in virtual machine
--------------------
sudo apt install python3-pip
pip install tk
